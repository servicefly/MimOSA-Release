## 1. Summary

M5.3 adds long-term recall: MimOSA can store past turns/notes as embeddings and
later answer "what were we talking about last week?"-style questions by
similarity rather than exact keywords. The whole pipeline is **local** and
degrades through three layers so it works on any machine — from a fully
featured box with Chroma + sentence-transformers down to a bare Python install
with zero optional dependencies.

Key properties:

1. **Three-layer graceful degradation** — Chroma → in-process fallback vector
   store; sentence-transformers → injected embedder → dependency-free
   `HashingEmbedder`.
2. **Local-only** — embeddings and vectors stay on disk (JSONL fallback persist
   or Chroma's local dir); no embedding API calls.
3. **Stable, testable** — `use_chroma=False` forces the fallback so tests are
   hermetic and deterministic.

## 2. What was delivered

### Semantic memory

| Capability | Detail |
|------------|--------|
| Store | `SemanticMemory` — Chroma collection when available, else `_FallbackVectorStore` (brute-force cosine, JSONL persistence) |
| Embeddings | injected embedder → `sentence-transformers` → `HashingEmbedder` (deterministic SHA1-bucketed, L2-normalized, no deps) |
| Record | `SemanticResult` dataclass (id, text, score, metadata) |
| Write | `add`, `add_turn` |
| Recall | `query`, `recall` (top-k by cosine similarity) |
| Lifecycle | `delete`, `reset`, `count` |
| Capability flags | `HAS_CHROMA`, `HAS_SENTENCE_TRANSFORMERS` |

> **Degradation note:** with no ML deps installed the `HashingEmbedder` is a
> *lexical* embedding (token-overlap), not a true semantic one. Recall still
> works and stays deterministic; installing `sentence-transformers` upgrades it
> to genuine semantic similarity with no code changes.

## 4. Privacy & resource notes

- Embeddings are computed and stored **locally** — no text or vector is sent to
  any embedding service.
- Gated by `PrivacySettings.semantic_memory`.
- Optional heavy deps (`chromadb`, `sentence-transformers`) are **commented** in
  `requirements.txt`; the core install stays light and the fallback keeps the
  feature working without them.
- Private turns can be excluded from the index by callers, keeping sensitive
  content out of long-term recall when desired.
